Gestión de Base de Datos

Administra los respaldos de tu base de datos de forma segura. Descarga, crea nuevos backups o restaura desde un archivo existente.

Descargar Backup

Descarga una copia de seguridad actual de la base de datos sin afectar los datos existentes.

Crear Nuevo Backup

Elimina la base de datos después de generar el respaldo (NO borra usuarios)

Subir Backup

Restaura la base de datos desde un archivo de backup existente. Selecciona un archivo .json válido.

Información Importante

  • Los backups se descargan automáticamente con la fecha actual en el nombre del archivo.
  • La opción "Crear Nuevo Backup" eliminará permanentemente todos los datos actuales. Pero NO borrara usuarios.
  • Solo se aceptan archivos de backup válido (.json).
